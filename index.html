Telegram Weather Bot

Application | 2022
Description:

Weather bot built with PYTHON.

The Telegram Weather Bot is a convenient tool for getting up-to-date weather information in your city. Written in Python, this bot uses "telebot", "requests" and "json" libraries to interact with users and fetch weather data from external sources.

Key features:

- Get current weather in specified city
- Provide forecast for upcoming days
- User-friendly Telegram interface
- This bot will become your indispensable assistant, always providing accurate and up-to-date weather data.

SPAM BOT

Application | 2022
Description:

This spam bot allows sending multiple messages to a user using a text file. It utilizes the PyAutoGUI module for on-screen text input and emulating the "Enter" key press. After a 5-second delay, it begins sending messages from a pre-created text file.

Key features:

- Automated text message spamming
- Uses PyAutoGUI for input simulation
- Reads messages from text files
- Customizable delay before execution
